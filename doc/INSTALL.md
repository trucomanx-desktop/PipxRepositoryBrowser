# pipx-repository-browser

Browse curated Python package repositories and manage installations with pipx.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/pipx_repository_browser/

```bash
pip install --upgrade pipx_repository_browser
```

or (recommended for desktop applications)

```bash
pipx install pipx_repository_browser
```


Using:

```bash
pipx-repository-browser
```

## Install from source
Installing `pipx-repository-browser` program

```bash
git clone https://github.com/trucomanx-desktop/PipxRepositoryBrowser.git
cd PipxRepositoryBrowser
pip install -r requirements.txt
cd src
python -m build
pip install dist/pipx_repository_browser-*.tar.gz
```
Using:

```bash
pipx-repository-browser
```

## Uninstall

```bash
pip uninstall pipx_repository_browser
```
